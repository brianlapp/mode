‚öôÔ∏è Configuration Options

üè¢ Property Codes

  • mff - ModeFreeFinds
  • mmm - ModeMarketMunchies
  • mcad - ModeClassActionsDaily
  • mmd - ModeMobileDaily

üìç Placement Options

  • thankyou - Show immediately on thank you pages
  • exit-intent - Show when user moves to leave
  • timed - Show after 10 seconds

üîÑ Frequency Options

  • session - Once per browser session
  • daily - Once per day
  • always - Every page load (testing only)

‚úÖ Deployment Best Practices

1

Start with Thank You Pages

Deploy on confirmation/thank you pages first - these have the highest engagement rates and won't interfere with lead capture.

‚úÖ Recommended: Place after form submission, on success pages, after email opt-ins
2

Use Session Frequency

Start with 'session' frequency to avoid overwhelming users while testing performance.

‚úÖ Best Practice: Session frequency prevents popup fatigue and maintains user experience
3

Test Before Wide Deployment

Test on a single thank you page first, monitor performance, then expand to other pages.

‚ö†Ô∏è Important: Monitor analytics for 24-48 hours before expanding deployment
4

Monitor Performance

Track popup impressions and clicks through the admin dashboard analytics.

‚úÖ Target Metrics: 15-25% click-through rate, $0.45+ CPL, positive user feedback

üîç Troubleshooting

Popup not showing?

  • Check browser console for errors (F12)
  • Verify property code matches your site
  • Try frequency: 'always' for testing
  • Clear browser cache and localStorage

Images not loading?

  • Check admin dashboard for campaign status
  • Verify campaigns are set to "Active"
  • Test on different devices/browsers

Need help?

Check the Admin Dashboard for campaign status and analytics.
